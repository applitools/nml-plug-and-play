# typescript-wdio-appium-nml-ios-multi-device-saucelabs

## Environment variables

- `APPLITOOLS_API_KEY`
- `SAUCE_USERNAME`
- `SAUCE_ACCESS_KEY`
- `SAUCE_REGION`
- `SAUCE_APP_ANALYTICSX`
- `SAUCE_APP_ACCESSIBILITY`
- `ACTIVE_APP`
- `DEVICE_NAME`
- `PLATFORM_VERSION`
- `FLOW`

## Run

```
npx wdio run ./wdio.ios.saucelabs.conf.ts
```
