# javascript-wdio-appium-nml-ios-perfecto

## Environment variables

- `APPLITOOLS_API_KEY`
- `PERFECTO_CLOUD_NAME`
- `PERFECTO_SECURITY_TOKEN`
- `PERFECTO_APP_ANALYTICSX`
