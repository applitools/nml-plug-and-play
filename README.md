# javascript-wdio-appium-nml-ios-multi-device-lambdatest

- `APPLITOOLS_API_KEY`
- `LT_USERNAME`
- `LT_ACCESS_KEY`
- `LT_APP_ANALYTICSX`
- `LT_APP_ACCESSIBILITY`
- `ACTIVE_APP`
- `DEVICE_NAME`
- `PLATFORM_VERSION`

