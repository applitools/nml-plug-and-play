# typescript-wdio-appium-nml-android-lambdatest

## Environment variables

- `APPLITOOLS_API_KEY`
- `LT_USERNAME`
- `LT_ACCESS_KEY`
- `LT_APP_ANALYTICSX`
- `ANALYTICSX_APP_PACKAGE`
- `ANALYTICSX_APP_ACTIVITY`
- `LT_APP_ACCESSIBILITY`
- `ACCESSIBILITY_APP_PACKAGE`
- `ACCESSIBILITY_APP_ACTIVITY`
- `ACTIVE_APP`
- `DEVICE_NAME`
- `PLATFORM_VERSION`
- `FLOW`

## Run

```
npx wdio run ./wdio.android.lambdatest.conf.ts
```
